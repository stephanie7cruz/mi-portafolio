ColegiosApp

Este Backend está creado usando .NET6, una API Rest y la conexión a base de datos se hizo utilizando Entity Framework. Para crear la base de datos en tu máquina local, debes modificar el (connection string) de nombre DbContext en el archivo appsettings.json con la ruta de tu base de datos local. Luego, debes ejecutar el comando Update-database desde la consola del administrador de paquetes para ejecutar todas las migraciones pendientes.
